# pdm - 생체신호 마이닝 (2023 2학기)
## Physiological Data Mining
---
* Physiological data
  - images & movies: X-ray, CT, MRI, PET, ...
  - signals: ECG, EEG, PPG, EMG, ...
  - text/tabular: 문진, 혈액검사, DNA, EHR, ...
* Data Mining
  - 데이터 수집 (Acquisition)
  - 데이터 전처리(Preprocessing)
  - 데이터시각화 (Visualization)
  - 데이터 분석 (Analysis)
    * 특징 추출 및 머신러닝
    * 딥러닝
    * AutoML
  - AI 모델 기반 WEB 서비스
    * streamlit
